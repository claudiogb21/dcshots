Setup inicial

1) Instalar las siguentes dependencias con npm:

"devDependencies": { "html-webpack-plugin": "^3.2.0", "webpack": "^4.5.0", "webpack-cli": "^2.0.14" }

2) crear el fichero webpack.config.js

Indicar el "entry", "output" y "plugins" principales. HtmlWebpackPlugin, ExtractTextPlugin (extrae los css generados los los loaders de sass y crea un archivo css que se incluirá luego en el template)

Luego crear los "Loaders" dentro de "rules". style-loader, css-loader, scc-loader.

ExtractTextPlugin docs

rules: [ { test: /\.scss$/, use: ExtractTextPlugin.extract({ fallback: 'style-loader', use: ['css-loader', 'sass-loader'], publicPath: './dist' }) } ]

3) instalar y configurar "webpack-dev-server"

Cambiar el script en package.json de: {"dev": "webpack -d --watch"}, a: {"dev": "webpack-dev-server -d"}


4) Configurar el "file-loader"

Para poder utilizar imágenes en los componentes, html y css de debe confgurar el "file-loader" dentro de modules -> rules:

{ test: /\.(jpe?g|png|gif|svg)$/i, use: [ { loader: 'file-loader', options: { name: '[name].[ext]', outputPath: 'img/', publicPath: 'img/' } } ] }

File-loader docs

5) Configuración de DevServer con Webpack

DevServer config

6) Instalar React y Babel

Primero instalar el loader para babel. Luego de instalar el resto de las dependencias, se debe crear un archivo ".babelrc" en la raíz:

{ "presets": ["es2015", "react"] }

React: Correr npm i -S react react-dom
Babel Loaders: Correr npm i --D babel-loader babel-core
Babel: Correr npm i -D babel-preset-env babel-preset-react
